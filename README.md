# TCAP API 

## Project versions

* Ruby - 2.7.0
* Rails - 6.0.3
* Postgres - 11

## Project Setup

1. clone repo
2. run `bundle install` to install dependencies.
3. run `chmod +x scripts/*.bash` to make scripts executables.
4. run `./scripts/install-hooks.bash` to add pre-commit and pre-push to your local git.
